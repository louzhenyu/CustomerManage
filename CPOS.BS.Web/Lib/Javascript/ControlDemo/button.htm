标准按钮控件
示例
代码
                    Ext.create('Jit.button.Button', {
                        renderTo: 'dvPlaceholder1'
                        , text: '新增'
                        , jitIsHighlight: false   //是否高亮
                        , jitIsDefaultCSS: false      //是否需要样式
                        , handler: function () {
                            alert('【新增】按钮被点击.');
                        }
                    });
